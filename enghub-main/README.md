# EngHub — Инженерная платформа (v5.1)

Веб-платформа для управления инженерными проектами проектного института. Вдохновлено дизайном Figma, Bitrix24 и Microsoft Teams.

## 🚀 Возможности

- **4 роли пользователей**: Администратор → ГИП → Руководитель отдела → Инженер
- **Управление проектами**: создание, архивирование, автоматический прогресс (% выполнения)
- **Канбан-доска задач**: 6 статусов с полным жизненным циклом и цветными приоритетами
- **⊕ Конференц-зал**: проектный видео-чат с историей сообщений и списком участников
- **🔍 Поиск и навигация**: breadcrumbs, глобальный поиск и фильтрация задач
- **Админ-панель**: управление пользователями, отделами, архивом
- **Тёмная/светлая тема**: современный адаптивный интерфейс* AI Orchestrator (Copilot) - Phase 5 UI is developed. Backend in progress.

### Запуск и Тестирование

Для полного E2E тестирования проекта был создан скрипт для генерации чистых тестовых пользователей во все отделы с правильными ролями.
*Запуск скрипта генерации пользователей:*
`node scripts/seed_test_users.mjs`

Все тестовые пользователи (например `gip_test@enghub.local`, `lead_dept3@enghub.local`) по умолчанию имеют пароль: **`123456`**.

## Стек технологий

| Компонент | Технология |
|-----------|-----------|
| Фронтенд | React 18 + TypeScript |
| Хостинг | Vercel |
| БД / Auth | Supabase (PostgreSQL) |
| Стили | Vanilla CSS (Figma Tokens, Flexbox, Grid) |
| Конференции | LiveKit (в процессе интеграции) |

## 📁 Структура проекта

```
src/
├── api/
│   └── supabase.ts          # API-хелперы (CRUD, Auth)
├── components/
│   ├── ui.tsx                # Базовые UI-компоненты
│   └── Notifications.tsx     # Система уведомлений (Toast)
├── pages/
│   ├── LoginPage.tsx         # Вход в систему
│   ├── AdminPanel.tsx        # Админка
│   └── ConferenceRoom.tsx    # Видео-конференции и чат
├── App.tsx                   # Основной интерфейс (Dashboard, Project, Tasks)
├── constants.ts              # Темы, статусы, роли
└── styles.css                # Дизайн-система (v5.1 Figma Redesign)
```

## 🔄 Жизненный цикл задачи

```
todo → inprogress → review_lead → review_gip → done
                                ↘ revision → inprogress
```

## 📝 Последние изменения

### v6.0 — Инженерные Расчёты (Calculation Engine)
- ✅ **Split-Pane UI**: Удобный каталог из более чем 90 расчётов распределен по маркам в левом меню для мгновенной навигации без зависаний и всплывающих окон.

- ✅ **Генерация Формул LaTeX**: Движок автоматически визуализирует теоретические формулы и их подстановку с использованием математического шрифта.
